### Как проверить программу

Перед запуском проверь, чтобы в папке с кодом лежал файл hosts.txt со списком сайтов.

1. Обычная проверка (Таблица)
Программа просто опрашивает каждый сайт по 3 раза.
Команда:
go run main.go -c 3 hosts.txt

Что будет: В консоли появится ровная таблица. Между попытками для каждого сайта будет пауза в 1 секунду.


2. Режим мониторинга (Бесконечный)
Программа постоянно следит за состоянием сайтов.
Команда:
go run main.go -monitor hosts.txt

Что будет: 
- Каждые 5 секунд будут выводиться новые строчки с текущим временем.
- В папке автоматически создастся файл monitor.log. Можно открыть его и увидеть, что все записи дублируются туда в реальном времени.


3. Правильное выключение
Пока идет мониторинг, нажми Ctrl+C.
Что будет: Программа не вылетит с ошибкой, а напишет «Завершение работы...», подождет все процессы и сама закроет файл лога.


4. Бонус на «отлично» (ICMP Ping)
Чтобы использовать реальный пинг, запусти терминал от имени Администратора.
Команда:
go run main.go -monitor -icmp hosts.txt

Что будет: Программа начнет работать через настоящий протокол ICMP.
